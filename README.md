# Cześć :)
Poniżej zamieszczam kilka informacji odnośnie rozwiązania zadania:

- ``` AppearanceFrequency``` -> wymagania nie zawierają warunku dla wartości ```2```, więc przyjąłem założenie, że będzie temu odpowiadać typ ```.low```
- ```SortMethod``` -> wymagania nie zawierały, według jakiego porządku ma być sorotowanie, przyjąłem, że będzie ```malejąco```
